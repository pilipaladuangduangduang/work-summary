# 智慧云个人中心

## 个人中心账号信息接口

```
{
	"url": "http://{ip:port}/app/smc_personal/info",
	
	"http-method": "get",
	
	"request-params": {
	},
	
	"response-data": {
            "result": {
                "exc_vip": true, # 是否是专属会员
                "nick_name": "test", # 昵称
                "icon": "test", # 头像
                "smc_vip": true, # 是否是智慧云会员
                "exc_vip_days": 0 # 专属会员过期的天数
            },
            "resultMsg": "success",
            "resultCode": 0,
            "success": true
        }
	
}
```
