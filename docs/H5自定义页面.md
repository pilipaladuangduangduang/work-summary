## 校验页面是否已经存在

```
{
	"url": "http://{ip:port}/cms/h5/check",
	
	"http-method": "get",
	
	"request-params": {
        "link": "xxxx"
	},
	
	"response-data": {
            "resultMsg": "操作成功",
            "resultCode": 0,
            "data": true # true表示存在则不允许下一步，false表示不存在则可以下一步
        }
	
}
```
