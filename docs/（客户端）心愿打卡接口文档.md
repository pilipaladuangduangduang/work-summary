# 数据类型

描述：

XxxForm 指的是前端提交给后端的数据模型

XxxModel 指的是后端返回给前端的数据模型

## HabitModel

| 名称               | 类型               | 描述
| :----------------- | :----------------- | :----------------- 
| id                 | string             | 习惯ID
| name               | string             | 响应信息
| selected           | boolean            | true 为默认选中，false 为不选中
| reward             | int                | 完成习惯获得的星星个数，默认为 1 个
