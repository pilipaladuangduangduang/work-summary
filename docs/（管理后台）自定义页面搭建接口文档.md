# 自定义页面新增

### 描述↓↓↓↓↓↓↓

用于新增自定义页面和原生页面（native）

### 注意事项↓↓↓↓↓↓↓

无

### 接口地址↓↓↓↓↓↓↓

http://120.26.4.86/cms/customize/add POST

### 接口开发进度↓↓↓↓↓↓↓

[OK](#开发状态字典表)

### 请求参数↓↓↓↓↓↓↓

| 名称               | 类型               | 必填               | 描述
| :----------------- | :----------------- | :----------------- | :----------------- 
| name               | string             | yes                | 自定义页面名称
| uri                | object             | yes                | 具体的页面内容，是一个大json
| type               | string             | yes                | 目前时这两种：h5、native 
| cover              | string             | no                 | 展示图片
| brief              | string             | no                 | 自定义页面描述
| remark             | string             | no                 | 备注

| 名称               | 类型               | 描述
| :----------------- | :----------------- | :----------------- 
| resultCode         | int                | 响应码，详细信息请点[这里](#返回码字典表)
| resultMsg          | string             | 响应信息
| data               | object             | null

# 数据类型

描述：

XxxForm 指的是前端提交给后端的数据模型

XxxModel 指的是后端返回给前端的数据模型

### CustomizeModel

| 名称               | 类型               | 描述
| :----------------- | :----------------- | :----------------- 
| id                 | string             | 自定义页面的唯一ID
| name               | string             | 自定义页面名称
| type               | string             | 自定义页面类型：h5、native
| href               | string             | 自定义页面的地址
| cover              | string             | 封面
| uri                | object             | 具体的内容，是一个大json
| brief              | string             | 页面描述
| remark             | string             | 备注
| create_user        | string             | 创建者
| modify_user        | string             | 修改者
| create_time        | datetime           | 创建时间
| modify_time        | datetime           | 修改时间
